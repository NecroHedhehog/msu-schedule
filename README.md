# MSU Schedule

Сервис расписания для студентов МГУ — автоматический сбор, Telegram-бот, веб-интерфейс.

Бот: [@msu_schedule_test_bot](https://t.me/msu_schedule_test_bot)

## Что делает

Парсеры ходят на сайты расписания факультетов МГУ, собирают данные по всем группам и курсам, складывают в единую базу. Telegram-бот даёт студентам доступ к расписанию: выбор группы кнопками, расписание на день/неделю с навигацией, персональный фильтр предметов по выбору.

### Возможности бота

- Выбор группы кнопками (направление → курс → группа) или текстом
- Расписание на сегодня / завтра / неделю
- Навигация по неделям (в пределах доступных данных)
- Фильтр предметов по выбору — студент отмечает свои, остальные скрыты
- Корректное время пар, включая МФК-слоты по средам
- В выходные автоматически показывает следующую неделю
- Нормализация ввода (русская/английская раскладка, ввод без префикса)
- Аналитика пользователей и действий (доступна админу через `/stats`)
- Настраиваемая рекламная система через `.env`

## Архитектура

```
┌─────────────┐     ┌──────────┐     ┌──────────────────┐
│  Парсеры    │────▶│  SQLite  │◀────│  Telegram-бот    │
│  (cron)     │     │          │     │  (aiogram)       │
└─────────────┘     │          │◀────┌──────────────────┐
                    └──────────┘     │  Веб [WIP]       │
                                    └──────────────────┘
```

Компоненты независимы: парсер упал — бот продолжает отдавать последние данные.

## Стек

Python 3.11, BeautifulSoup4, aiogram 3, SQLite (WAL), systemd на VPS.

## Структура

```
core/
  config.py        — конфигурация из .env, время пар
  database.py      — схема БД, CRUD, фильтрация, аналитика
parsers/
  base.py          — базовый класс парсера
  socio.py         — соцфак (cacs.socio.msu.ru)
  config/          — конфиги парсеров (не в репо)
bot/
  main.py          — хендлеры, навигация, inline-кнопки, трекинг
  formatting.py    — форматирование сообщений
run_parser.py      — точка входа парсера
run_bot.py         — точка входа бота
```

## Текущий охват

- **Соцфак** (`cacs.socio.msu.ru`) — 46 групп, бакалавриат + магистратура, все направления и курсы
- **ФГУ** (`cacs.spa.msu.ru`) — структура разобрана, парсер в планах

## Схема данных

Основные таблицы: `faculties`, `groups_`, `lessons`, `subscriptions`, `user_subjects`. Аналитика: `users` (Telegram-профили), `activity_log` (действия). Парсер при повторном запуске обновляет существующие записи, не дублирует.

На одну пару может приходиться несколько предметов по выбору — бот позволяет студенту отметить свои и фильтрует остальные.

## Технические заметки

Расписание соцфака отдаётся помесячно, кодировка cp1251. Навигация четырёхуровневая: отделение → направление → год набора → группа.

По средам пары 4–5 — МФК-слоты с отдельным временем (15:10–16:40, 17:00–18:30), парсер определяет день недели и подставляет корректное время.

ФГУ/юрфак — другая платформа (div-блоки вместо таблиц, UTF-8, `/ttablestd/`). Отдельный адаптер, но формат данных совместим.
